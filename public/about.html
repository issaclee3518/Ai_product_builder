메모 화이트보드 소개

issac.store · 브라우저 기반 2D / 3D 스케치 도구

메모 화이트보드는 별도의 설치나 회원 가입 없이, 웹 브라우저만으로 아이디어를 빠르게 스케치할 수 있는 도구입니다. 평면(2D) 그림판과 공간(3D) 배치 모드를 전환하며 간단한 메모부터 도형 구성까지 다룰 수 있습니다.

왜 만들었나요?

회의 중 떠오른 구조, 화면 레이아웃, 간단한 다이어그램을 바로 그리기 위해 만들었습니다. 복잡한 디자인 툴을 열지 않고도, 브라우저 탭 하나에서 선을 긋고 도형을 놓고 색을 입힐 수 있도록 구성했습니다.

주요 기능

  • 2D 화이트보드 — 자유롭게 그리고, 홀드하면 선·원·사각형·삼각형 등으로 정리되는 퀵셰이프를 지원합니다.
  • 3D 화이트보드 — 도형을 배치하고 이동·회전·크기 조절하며 카메라를 조작할 수 있습니다.
  • 색상·굵기 조절 — 도구막대에서 선 색과 두께를 바로 바꿀 수 있습니다.
  • 이미지 저장 — 작업한 보드를 PNG로 내려받을 수 있습니다.

이런 분께 적합합니다

  • 빠른 스케치·메모가 필요한 학생과 개발자
  • 회의에서 UI 와이어나 구조를 대략 잡고 싶은 분
  • 설치 없이 공유 가능한 가벼운 보드가 필요한 분
